# Lyrics Autoscroll MVP

Este es un proyecto simple de una sola página ("Single Page Application") que no requiere instalación ni compilación.

## Cómo ejecutar el proyecto (Ver la web)

Como es un archivo HTML estándar con JavaScript "Vanilla" (sin frameworks), tu navegador ya sabe cómo interpretarlo directamente.

1. **Opción 1 (Doble clic)**: Busca el archivo `index.html` en tu carpeta y haz doble clic sobre él. Se abrirá en tu navegador predeterminado.
2. **Opción 2 (Arrastrar)**: Abre tu navegador (Chrome, Safari, Firefox) y arrastra el archivo `index.html` dentro de una pestaña.

## Cómo ver tus cambios

El ciclo de desarrollo es muy rápido:

1. Abre `index.html` en tu editor de código (VS Code, etc).
2. Haz un cambio (por ejemplo, cambia el color de fondo en el CSS).
3. **Guarda** el archivo (Cmd+S o Ctrl+S).
4. Ve a la pestaña del navegador donde tienes abierta la web.
5. **Recarga** la página (Cmd+R, F5 o el botón de refrescar).

¡Listo! Verás tus cambios reflejados inmediatamente. No hace falta "compilar" ni reiniciar nada.
